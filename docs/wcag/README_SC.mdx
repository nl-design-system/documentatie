# Structuur van een artikel over een SC

Metadata in frontmatter: 
- titel (unieke tekst eerst, bijv. 1.4.3 - Contrast - NL Design Systems), 
- description (max 150 chars) 
- keywords (zoek welke woorden worden gebruikt om dit onderwerp te vinden)

## Uitleg
Introductietekst, vanuit een herbruikbaar markdown bestand (docs/wcag/summaries/xx.mdx) (aanpassen in dezelfde branch)

Gedetailleerd vervolg op introductietekst (wek interesse voor dit onderwerp). Momenteel staat veel nuttige info onder "hoe te testen".
