Bij elk veld in een formulier staat een zichtbaar label dat beschrijft wat de bezoeker moet invullen.

Bijvoorbeeld: "Voornaam", "E-mailadres", "Postcode". Het label blijft zichtbaar tijdens het invullen en mag ook een duidelijk icoon zijn. Waar het invoerformaat niet vanzelfsprekend is, staat er een instructie bij het veld. Zo kan de bezoeker het formulier in één keer goed invullen.

Het gaat hier om wat de bezoeker op het scherm ziet. De koppeling tussen label en veld in de code valt onder succescriterium 1.3.1, en foutmeldingen vallen onder succescriterium 3.3.1.
