Wanneer iemand met hulpsoftware een pagina van begin tot eind doorleest, moet de informatie in een logische en begrijpelijke volgorde staan. De leesvolgorde wordt bepaald door de HTML-broncode, niet door de visuele lay-out. CSS kan elementen visueel verplaatsen, maar verandert de volgorde in de code niet. Dat verschil is waar het misgaat.
