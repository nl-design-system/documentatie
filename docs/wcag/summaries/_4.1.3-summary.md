Een statusbericht is een melding die op de pagina verschijnt als reactie op een actie of een verandering, zonder dat de focus verspringt. Bijvoorbeeld: een bevestiging dat een product is toegevoegd aan de winkelwagen, het aantal zoekresultaten onder een zoekveld, of een waarschuwing dat een sessie bijna verloopt.

Markeer deze meldingen in de code als live region: een element met `role="status"`, `role="alert"` of `aria-live`. Een [screenreader](/woordenlijst/#screenreader) leest ze dan automatisch voor. Zonder die markering verschijnt het bericht wel op het scherm, maar merkt een screenreader er niets van.

Een melding die de focus krijgt, is geen statusbericht; die valt onder [3.3.1 Foutidentificatie](/wcag/3.3.1/). Componenten met een eigen rol, zoals een dialoog, vallen onder [4.1.2 Naam, rol, waarde](/wcag/4.1.2/).
