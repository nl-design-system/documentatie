Als je met het toetsenbord door een webpagina navigeert, moet je altijd kunnen zien welk element op dat moment de focus heeft. Er moet een zichtbare indicator zijn, meestal een omlijning of markering, die aangeeft waar je je bevindt. Zonder die indicator navigeer je blind. Het gaat om de focusindicator die verschijnt bij toetsenbordnavigatie, niet bij focus die door een script is gestuurd. Denk aan een element dat focus kreeg na het klikken op een skiplink.
