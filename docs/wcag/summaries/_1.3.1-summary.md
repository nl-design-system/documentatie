Alle bezoekers van een website moeten dezelfde informatie kunnen gebruiken. Wat visueel wordt getoond aan ziende bezoekers, zoals de informatie zelf, de structuur en de samenhangen tussen onderdelen, moet ook in de code van de pagina staan. Zo kunnen mensen die hulpsoftware gebruiken, zoals een schermlezer, deze informatie laten voorlezen.
