Bij een vooraf opgenomen video waarin wordt gesproken biedt een website alternatieven voor wie de beelden niet kan zien. Dit succescriterium geeft twee keuzes: je voegt een audiodescriptie toe, óf je biedt een volledig tekstalternatief aan dat zowel de dialoog als de visuele gebeurtenissen beschrijft. Dit is een niveau A-criterium, dus elke website moet aan één van deze twee opties voldoen.
