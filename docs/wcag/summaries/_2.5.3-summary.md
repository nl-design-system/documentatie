Als een interactief element een zichtbaar label heeft — de tekst die je op het scherm ziet bij een knop, link of formulierveld — dan moet die zichtbare tekst ook onderdeel zijn van de toegankelijke naam van dat element. De toegankelijke naam is wat screenreaders en spraakbesturing gebruiken om het element de functie van het element te begrijpen.
