Als een interactief element een zichtbaar label heeft, moet die zichtbare tekst ook onderdeel zijn van de toegankelijke naam van dat element. De zichtbare tekst is wat je op het scherm ziet bij een knop, link of formulierveld. De toegankelijke naam is wat screenreaders en spraakbesturing gebruiken om de functie van het element te begrijpen.
