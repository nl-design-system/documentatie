# Baseline toegankelijkheids-ondersteuning

Om vast te kunnen stellen of iets toegankelijk is en voldoet volgens WCAG 2.2, niveau AA moet je weten welke software wordt ondersteund. Voor NL Design System zijn dat de volgende combinaties van de volgende besturingssystemen, browsers en screenreaders:

Windows

- Firefox + NVDA
- Chrome + NVDA
- Chrome + JAWS

macOS

- Safari + VoiceOver

Android

- Chrome + Talkback

iOS

- Safari + VoiceOver

## Uitleg

WCAG is een set richtlijnen die niet voorschrijft hoe je precies toegankelijke producten maakt. Het is techniekonafhankelijk. Wanneer je een techniek gebruikt die niet goed ondersteund wordt, kan het gebeuren dat een component toegankelijk lijkt, maar het voor gebruikers niet is.

Daarom stellen we vast welke technologieen we ondersteunen. Deze gebruiken we vervolgens om de componenten mee te testen. Zodoende weten gebruikers, maar ook toegankelijkheidsonderzoekers met welke software de componenten van NL Design System werken.
