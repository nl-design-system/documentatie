# Baseline toegankelijkheids-ondersteuning

NL Design System ondersteunt de twee meest recente versies van de volgende combinaties van de volgende besturingssystemen, browsers en screenreaders:

Windows

- Firefox + NVDA
- Chrome + NVDA
- Chrome + JAWS

macOS

- Safari + VoiceOver

Android

- Chrome + Talkback

iOS

- Safari + VoiceOver

## Uitleg

Om vast te kunnen stellen of iets toegankelijk is en voldoet volgens WCAG 2.2, niveau AA moet je weten welke software wordt ondersteund. De Web Content Accessibility Guidelines (WCAG) zijn een set richtlijnen die niet voorschrijven hoe je precies toegankelijke producten maakt. Het is techniekonafhankelijk. Wanneer je een techniek gebruikt die niet goed ondersteund wordt, kan het gebeuren dat een component toegankelijk lijkt, maar het voor gebruikers niet is.

Daarom stellen we vast welke technologieen we ondersteunen. Deze gebruiken we vervolgens om de componenten mee te testen tijdens het ontwikkelen van de componenten. Zodoende weten gebruikers, ontwikkelaars en ook toegankelijkheidsonderzoekers met welke software de componenten van NL Design System toegankelijk zijn.
