## Best practice

### Geef elke navigatie een toegankelijke naam

Als er meerdere navigaties op een pagina staan, kunnen screenreadergebruikers niet horen wat het verschil is. Geef elke navigatie een toegankelijke naam.

**Hoe te testen:** controleer of elk `<nav>`-element een `aria-label` of `aria-labelledby` heeft als er meerdere navigaties op de pagina staan.

```mdx
<Guideline appearance="dont" title="Twee navigaties zonder naam.">

```html
<nav>...</nav>
<nav>...</nav>
```

</Guideline>

<Guideline appearance="do" title="Navigaties met een naam.">

```html
<nav aria-label="Hoofdnavigatie">...</nav>
<nav aria-label="Footernavigatie">...</nav>
```

</Guideline>
```

**Wie kan dit toepassen:** een developer voegt een `aria-label` toe aan elk `<nav>`-element.
