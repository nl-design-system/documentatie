# Alt-tekst keuzehulp

[![npm version](https://img.shields.io/npm/v/@nl-design-system-community/clippy-alt-text-wizard)](https://www.npmjs.com/package/@nl-design-system-community/clippy-alt-text-wizard)
[![GitHub](https://img.shields.io/badge/github-repo-blue?logo=github)](https://github.com/nl-design-system/editor)

Twijfel je over welke alt-tekst je moet gebruiken? De alt-tekst keuzehulp stelt je een paar korte vragen over je afbeelding en geeft direct advies over of alt-tekst nodig is en wat je moet schrijven. Zo zorg je ervoor dat iedereen je content kan begrijpen, ook met hulptechnologie zoals een schermlezer.
