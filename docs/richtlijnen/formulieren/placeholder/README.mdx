# Richtlijnen NL Design System voor placeholders in een formulier

Een placeholder geeft de gebruiker een korte hint over het verwachte type gegevens dat in het formulierveld moet worden.

Een [label](/richtlijnen/formulieren/labels/) vertelt **wat** je moet invullen en een [placeholder](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/placeholder) vertelt wat voor **soort gegevens** in een veld wordt verwacht. Voor een e-mailveld kan het label bijvoorbeeld "Jouw e-mailadres" zijn en de placeholder "naam@voorbeeld.com".
