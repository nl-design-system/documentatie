# Richtlijnen NL Design System voor fieldsets in een formulier

Groepeer formuliervelden die bij elkaar horen in een `<fieldset>` met een beschrijvende `<legend>`.
Het voordeel hiervan is dat je zo de velden visueel, maar ook in de code aan elkaar koppelt.

De legend dient als de naam van de groep.

```html
<fieldset>
  <legend>Kies je maat</legend>
  <input id="size-s" name="size" type="radio" value="small" />
  <label for="size-s">Small</label>
  <input id="size-m" name="size" type="radio" value="medium" />
  <label for="size-m">Medium</label>
  <input id="size-l" name="size" type="radio" value="large" />
  <label for="size-l">Large</label>
</fieldset>
```
