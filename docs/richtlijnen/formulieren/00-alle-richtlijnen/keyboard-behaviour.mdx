# Zorg dat het formulier werkt met een toetsenbord

Een formulier moet goed werken en goed te begrijpen zijn voor een toetsenbordgebruiker. Hierbij is van toepassing:

- [Maak de toetsenbordfocus goed zichtbaar](/richtlijnen/formulieren/alle-richtlijnen/visueel-ontwerp/#maak-toetsenbord-focus-goed-zichtbaar) op de pagina Visueel Ontwerp van formulieren.
- Gebruik geen positieve tabindex.
