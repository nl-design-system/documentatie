Als het label van een Text input uit een afbeelding bestaat, dan heeft deze afbeelding een goed tekstalternatief, zodat de Text input een naam heeft die duidelijk maakt waar de het invoerveld voor dient.

Het beste is om altijd visueel een tekst te tonen naast of onder het icon. Icons zijn immers niet altijd voor iedereen duidelijk. Een tekstalternatief voor het icon is niet nodig als er tekst in beeld is. In dat geval kan het icon verborgen worden, bijvoorbeeld met een leeg `alt`-attribuut, of met `aria-hidden`.
