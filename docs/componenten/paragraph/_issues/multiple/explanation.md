De Paragraph bevat meerdere alinea's die zijn gescheiden met witregels. Hierdoor is er in code geen onderscheid tussen de alinea's. Een screenreader mist daardoor natuurlijke pauze-momenten in de tekst tijdens het voorlezen. Daarnaast kan een screenreadergebruiker niet met een sneltoets naar de volgende paragraaf gaan, omdat de teksten niet zijn opgemaakt met losse Paragraphs.
