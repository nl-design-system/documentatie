# NL Design System Baseline: augustus 2025

<Paragraph lead>
  NL Design System gebruikt een Baseline zodat onze websites en webapplicaties werken in de belangrijkste browsers en
  hulpmiddelen. Dit is het overzicht van de combinaties van software waar het kernteam vanaf augustus in test.
</Paragraph>

Technologie verandert continue, dus we gebruiken een Baseline om te laten zien welke softwareversies het Kernteam op een bepaald moment in test. Lees meer over onze aanpak op [NL Design System Baseline](http://nldesignsystem.nl/baseline/).

## Software om mee te testen

De volgende software en combinaties van software werken goed om functionaliteit en WCAG 2.2 successcriteria te testen.

De laatste versies van software op 22 augustus 2025 zijn:

- [Chrome 139](https://chromereleases.googleblog.com/2025/08/stable-channel-update-for-desktop_19.html) of [Edge 139](https://learn.microsoft.com/en-us/deployedge/microsoft-edge-relnote-stable-channel#version-13903405111-august-21-2025)
- Chrome 139 of Edge 139 met de [screenreader](/woordenlijst/#screenreader) [JAWS 2025](https://support.freedomscientific.com/Downloads/JAWS/JAWSWhatsNew?version=2025) op [Windows 11](https://www.microsoft.com/en-us/software-download/windows11)
- Chrome 139 of Edge 139 met de screenreader [NVDA 2025.2](https://www.nvaccess.org/post/nvda-2025-2/) op Windows 11
- [Chrome 139 op Android](https://chromereleases.googleblog.com/2025/08/chrome-for-android-update_01416888669.html)
- Chrome 139 op Android met de screenreader [Talkback 16.0](https://support.google.com/accessibility/android/answer/16294093?hl=en&ref_topic=10601875&sjid=15881667813641533001-EU)
- Chrome 139 op Windows 11 of [macOS 15.6 Sequoia](https://developer.apple.com/documentation/macos-release-notes/macos-15_6-release-notes)
- [Firefox 142](https://www.firefox.com/en-US/firefox/142.0/releasenotes/)
- Firefox 142 met de screenreader NVDA 2025.2 op Windows 11
- Safari [18.6](https://developer.apple.com/documentation/safari-release-notes/safari-18_6-release-notes) op [iOS 18.6](https://developer.apple.com/documentation/ios-ipados-release-notes/ios-ipados-18_6-release-notes)
- Safari 18.6 op iOS 18.6 met de screenreader VoiceOver
- Safari 18.6 op macOS 15.6
- Safari 18.6 op macOS 15.6 met de screenreader VoiceOver
