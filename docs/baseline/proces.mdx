# Nieuwe baseline bepalen

<Paragraph lead>NL Design System</Paragraph>

## Besturingssystemen

Bepaal de belangrijkste besturingssystemen voor desktop en voor mobiele apparaten, op basis van wereldwijd marktaandeel in de [RUM Archive](https://rumarchive.com/insights/#os-marketshare). Selecteer de besturingssystemen met meer dan 10% marktaandeel.

De laatst keer waren dat:

- Desktop:
  - Windows
  - macOS
- Mobiel:
  - iOS
  - Android

## Browsers

Bepaal de belangrijkste browsers voor desktop en mobiele apparaten, op basis van wereldwijd marktaandeel in de [RUM Archive](https://rumarchive.com/insights/#os-marketshare) en de [WebAIM Screen Reader survey](https://webaim.org/projects/screenreadersurvey10/#browsers). Gebruik de browsers met meer dan 10% marktaandeel in één of beide bronnen.

De laatst keer waren dat:

- Chrome
- Edge
- Safari
- Firefox (meer dan 10% in de WebAIM Screen Reader survey)

### Browser-combinaties

Bepaal welke screenreaders het meest gebruikt worden, en in welke combinatie tussen browser en screenreader.

Werk op basis van de meest recente Screen Reader User Survey in [het overzicht van WebAIM projecten](https://webaim.org/projects/). De laatste keer was dat [Screen Reader User Survey #10](https://webaim.org/projects/screenreadersurvey10/).

Kies de belangrijkste browser-screenreader-combianties op basis van de [Screen Reader / Browser Combinations in de meest recente Screen Reader User Survey](https://webaim.org/projects/screenreadersurvey10/#browsercombos).

Selecteer de combinaties die door meer dan 10% van de respondenten genoemd zijn. De laatst keer waren dat:

- JAWS with Chrome
- NVDA with Chrome
- JAWS with Edge
- NVDA with Firefox

Gebruik ook de combinaties die de default software zijn op de belangrijkste besturingssystemen, die een significant aandeel hebben. De laatst keer waren dat:

- VoiceOver voor Safari voor macOS heeft 7% aandeel, dat is significant.
- Windows Narrator voor Microsoft Edge voor Windows komt niet in de lijst voor.

Gebruik voor mobiele besturingssystemen de combinaties die de standaard software zijn op de belangrijkste besturingssystemen zijn. De laatst keer waren dat:

- VoiceOver voor Safari voor iOS
- Talkback voor Chrome voor Android

Op [Appt.org kun je statistieken vinden](https://appt.org/en/stats) hoe groot het percentage is van VoiceOver

<SpotlightSection>

</SpotlightSection>
