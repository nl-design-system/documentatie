# Form-control tokens in het basisthema

<Paragraph lead>
  Binnen de meeste thema's hebben formulierelementen zoals invoervelden, checkboxes en dropdowns veel overlap qua stijl.
  Ze gebruiken vaak dezelfde kleuren, borders en typografie voor dezelfde status (bv. focus, hover, invalid, etc.).
  Daarom defineert het basisthema een aantal design tokens die gedeeld worden door de verschillende form-componenten. Op
  deze pagina staan al deze gedeelde tokens en waar ze voor gebruikt worden.
</Paragraph>
