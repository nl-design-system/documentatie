# Spacing tokens in het basisthema

<Paragraph lead>
  Het basisthema bevat verschillende soorten tokens om aan te geven hoeveel ruimte (horizontaal en verticaal) er tussen
  (margin) of binnen (padding) elementen bewaard moet worden. Op deze pagina staat waar elke soort gebruikt voor wordt
  en welke tokens daarin zitten.
</Paragraph>
