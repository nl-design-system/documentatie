## Border-color tokens

Alle beschikbare kleuren zijn verwerkt in de [algemene kleurtokens](kleuren) dus er zijn geen aparte tokens voor border-color. Als je een kleur wil kiezen voor een border bepaal je eerst in welke kleurcategorie (bv. text, disabled, primary, etc) het element waar de border op komt het beste past. Als je een categorie gekozen hebt kun je daar een van de border tokens uit pakken (border-1, border-2 of border-3).
