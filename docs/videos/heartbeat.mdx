# Heartbeats

Het kernteam van NL Design System vertelt 2-wekelijks wat de laatste stand van zaken is. Daarnaast laten we organisaties aan het woord die met/aan het NL Design System werken.
Sinds kort nemen we elke sessie op. Hieronder maken we deze sessies beschikbaar om op elk gewenst moment zelf terug te kijken.

Wil je er (online) bij zijn, heb je vragen of wil je zelf iets vertellen of presenteren tijdens een van de heartbeats? Stuur ons dan een mailtje op <a href="mailto:nldesignsystem@ictu.nl">nldesignsystem@ictu.nl</a>
