# Bedankt voor je aanmelding voor de NL Design System Community

Welkom, fijn dat je erbij bent. We hopen je snel (weer) eens te zien bij een Heartbeat, samenwerkdag of bijeenkomst.

Handige links:

- Praat mee op [Slack](/slack) in het kanaal `#nl-design-system` Stel hier je vragen of volg de gesprekken.
- [Bekijk bijeenkomsten waar je aan mee kunt doen](/community/events/overzicht).
- [Kijk naar code op GitHub](https://github.com/nl-design-system/).
- Deel onze [factsheet voor managers](https://nldesignsystem.nl/factsheets/managers/) of [Over NL Design System](https://nldesignsystem.nl/project/over-nl-design-system) met collega's.
- Bekijk onze [video's op YouTube](https://www.youtube.com/@NLDesignSystem/), bijvoorbeeld van Design Systems Week of eerdere Heartbeats.
