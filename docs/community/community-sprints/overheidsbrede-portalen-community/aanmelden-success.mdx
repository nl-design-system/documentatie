# Bedankt!

Welkom bij de Overheidsbrede Portalen Community, fijn dat je erbij bent!

Handige links:

- [Word lid van het #nl-design-system-developers Slack kanaal](https://praatmee.codefor.nl) op CodeForNL, daar kun je meepraten, vragen stellen of beantwoorden of gesprekken volgen.
- [Bekijk de code op GitHub](https://github.com/nl-design-system/overheidsbrede-portalen-community).
- Bekijk onze [video's in de Overheidsbrede Portalen Community Sprint playlist op YouTube](https://www.youtube.com/playlist?list=PLnAxFMscDU3F1VYQxlJLpmV-Z-FQ6hCU_) en abonneer je op ons kanaal.
- Vind alle relevante linkjes en informatie op de [Overheidsbrede Portalen Community pagina](https://nldesignsystem.nl/community/community-sprints/overheidsbrede-portalen-community).

Elke twee weken heeft de Overheidsbrede Portalen Community een afstemming in het #nl-design-system-developers kanaal om aan de NL Design System adoptie binnen mijn omgevingen te werken.
