# Heartbeat

In de Heartbeat vertelt het kernteam van NL Design System elke twee weken wat de laatste stand van zaken is. Daarnaast laten we organisaties aan het woord die met/aan het NL Design System werken.

- **Tijdstip**: Elke even week, op dinsdag om 13:00 uur.
- **Locatie**: In de Microsoft Teams-meeting van de Heartbeat, <a href="/events/heartbeat/deelnemen">klik hier om deel te nemen</a>.
- **Kalenderbestand**: <a href="/heartbeat/heartbeat.ics">Download het kalenderbestand</a>.

## Ben je erbij?

Iedereen kan aansluiten!

De Heartbeat vindt plaats op Microsoft Teams, deze sessies zijn publiek toegankelijk.

Door je aan te melden kunnen wij je op de hoogte houden van Heartbeats. Je ontvangt dan ook direct een kalenderbestandje, zodat je de Heartbeat makkelijk aan je agenda kan toevoegen.
