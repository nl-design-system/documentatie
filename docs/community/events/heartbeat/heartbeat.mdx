# Heartbeat

In de Heartbeat vertelt het kernteam van NL Design System elke twee weken wat de laatste stand van zaken is. Daarnaast laten we organisaties aan het woord die met/aan het NL Design System werken.

## Ben je erbij?

Iedereen kan aansluiten!

De Heartbeat is elke even week, op dinsdag om 13:00 uur <a className="utrecht-link utrecht-link--html-a" href="/events/heartbeat/deelnemen">in Microsoft Teams</a>. De sessies zijn publiek toegankelijk en worden opgenomen.

Door je aan te melden kunnen wij je op de hoogte houden van Heartbeats. Je ontvangt dan ook direct een kalenderbestandje, zodat je de Heartbeat makkelijk aan je agenda kan toevoegen.

Werk jij met of aan het NL Design System? Dan kun je zelf ook iets presenteren! Lees meer over hoe dat werkt op <Link href="/events/heartbeat/zelf-iets-vertellen">zelf iets vertellen in de Heartbeat</Link>.
