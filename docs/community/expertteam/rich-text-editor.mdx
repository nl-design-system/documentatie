# Rich Text Editor

Momenteel is de Clippy Rich Text Editor volop ein ontwikkeling door het NL Design System Expertteam. De editor is een toegankelijke, framework-onafhankelijke teksteditor gebouwd met web components en TipTap voor het Nederlandse digitale overheidslandschap, welke zal voldoen aan ATAG en WCAG 2.2 Level AA-normen.

## Web Component (Lit)

De Clippy Rich Text Editor is beschikbaar als web component. Importeer de dependency en gebruik `<clippy-editor>Voorbeeld tekst</clippy-editor>` in je HTML om de editor te integreren.

[<img src="https://badge.fury.io/js/@nl-design-system-community%2Feditor.svg" alt="" style={{verticalAlign: 'middle'}} /> @nl-design-system-community/editor](https://www.npmjs.com/package/@nl-design-system-community/editor)

## React Component

De Clippy Rich Text Editor is tevens beschikbaar als React component. Importeer `ClippyEditor` in je React component.
