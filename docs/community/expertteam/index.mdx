# Expertteam

In september 2025 is het expertteam opgestart. Zij houden zich bezig met twee deelprojecten van NL Design System:

<CardGroup>
  <Card component="section" appearance="medium">
    <CardContent>
      <div>
        <Heading3>Theme Wizard</Heading3>
        <Paragraph>
          De theme wizard is een tool waarmee je NL Design System thema's kunt samenstellen door bestaande websites te
          analyseren en kleuren en lettertypes te selecteren. De gegenereerde design tokens en het thema tonen direct in
          een live preview de NL Design System componenten.
        </Paragraph>
      </div>
      <Paragraph>
        <Link to="/community/expertteam/theme-wizard">Lees verder over de Theme Wizard</Link>
      </Paragraph>
    </CardContent>
  </Card>

    <CardContent>
      <div>
        <Heading3>Rich Text Editor</Heading3>
        <Paragraph>
          Een toegankelijke, framework-onafhankelijke teksteditor gebouwd met web components en TipTap voor het
          Nederlandse digitale overheidslandschap, welke zal voldoen aan WCAG 2.2 Level AA-normen.
        </Paragraph>
      </div>
      <Paragraph>
