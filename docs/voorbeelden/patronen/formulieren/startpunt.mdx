# Startpunt van een meer-stappen-formulier

Het startpunt van een meer-stappen-formulier kan verschillen. Het startpunt van een meer-stappen-formulier dat we aan eindgebruikers hebben voorgelegd was een contactpagina.

![Ontwerp van een contactpagina als startpunt voor een meer-stappen-formulier](https://raw.githubusercontent.com/nl-design-system/documentatie/assets/voorbeeld_formulier_contact-pagina.png)

## Inzichten uit onderzoek

- Vrijwel elke respondent geeft aan dat ze in eerste instantie zouden bellen, of eventueel langsgaan.
- Het telefoonnummer wordt verwacht op de contactpagina. Daarnaast zoekt men het telefoonnummer ook in de footer.
- Een enkele respondent geeft aan misschien toch het formulier te proberen zodat ze haar vraag goed kan formuleren.

<VideoPlayer videoId="TiDvcgjYVDw" />

- Vrijwel elke respondent weet de link 'Start contactformulier' te vinden. Het werkwoord 'Start' gevolgd door het type formulier lijkt dus goed te werken.
