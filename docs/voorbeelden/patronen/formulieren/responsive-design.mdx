# Responsive design van een formulier

Net als de rest van een website dient een formulier op elk schermformaat goed te werken. Hieronder verzamelen we informatie rondom responsive design.

![Ontwerp van een responsive formulier](https://raw.githubusercontent.com/nl-design-system/documentatie/assets/voorbeeld_formulier_visual-design.png)

## Inzichten uit onderzoek

Meerdere respondenten geven aan het formulier normaal gesproken op hun smartphone of tablet in te vullen. Dit biedt voordelen vanwege de automatische suggesties bij het typen van tekst. Of zelfs de mogelijkheid om gebruik te maken van spraak naar tekst functionaliteit. Ook merkt een respondent op dat het scrollen op een tablet beter voor haar werkt in verband met haar beperking.
