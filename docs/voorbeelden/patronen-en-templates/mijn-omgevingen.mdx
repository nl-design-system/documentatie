# Mijn omgevingen

We verzamelen patronen en templates rondom mijn omgevingen via [GitHub Discussions](https://github.com/orgs/nl-design-system/discussions/categories/mijn-omgevingen).

[De overkoepelende GitHub Discussion voor mijn omgevingen](https://github.com/orgs/nl-design-system/discussions/386) is een goed startpunt om te beginnen.

Heb je inzichten uit onderzoek of vraagstukken die je wilt delen? Voeg ze toe aan de [GitHub Discussions](https://github.com/orgs/nl-design-system/discussions/categories/mijn-omgevingen). Misschien kan een andere organisatie jouw vraagstuk meenemen in toekomstig onderzoek.

Mis je een patroon of template? Voeg deze zelf toe als GitHub Discussion of [neem contact op met het kernteam](/project/kernteam)
