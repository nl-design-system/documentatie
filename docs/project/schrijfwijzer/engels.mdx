# Richtlijnen voor Engelse teksten

<Paragraph lead>
  Sommige pagina's op de website zijn Engelstalig, omdat ze bedoeld zijn voor een internationaal publiek. Veel
  softwaredocumentatie is ook alleen in het Engels beschikbaar, vanwege de Standard for Public Code.
</Paragraph>

## Consistent met API's

Schrijf teksten in Amerikaans-Engels, zodat de teksten consistent zijn met code van het design system. Bijvoorbeeld: "color" in plaats van "colour".

## Telefoonnummers

Gebruik de internationale notatie voor telefoonnummers in Engelse teksten, zodat ze ook werken buiten Nederland.

- +31 234 56 78 90
- +31 20 623 45 67
- +31 6 53 45 67 89
