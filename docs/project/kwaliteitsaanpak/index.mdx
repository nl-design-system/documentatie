# Kwaliteitsaanpak

<Paragraph lead>
  Werk je in opdracht van NL Design System? Gebruik dan deze aanpak om veilige software van hoge kwaliteit te maken.
  Wanneer je in de community werkt, dan heb je misschien ook veel aan deze aanpak.
</Paragraph>

## Overzicht

Voor je aan de slag gaat, is het belangrijk om te weten hoe je veilig je werk kunt doen:

- [Veilige werkomgeving](./veilige-werkomgeving)
- [Veilig werken](./veilig-werken)

Alle taken, ideeën en processen worden vastgelegd in de Product Backlog. De product owner bepaalt de prioriteiten, en die werkzaamheden worden uitgevoerd in Agile Sprints. Lees de details:

- [Product Backlog](./product-backlog)
- [Definition of Ready](./definition-of-ready)
- [Agile sprint](./agile-sprint)
