# Veilige werkomgeving

<Paragraph lead>
  Zorg voor een veilige werkomgeving, zodat je met open communicatie en effectieve samenwerking het beste resultaat kan
  bereiken. De gedragscode maakt duidelijk wat hier voor nodig is, en geeft het vertrouwen dat er een plan is voor
  handhaving. Controleer periodiek dat iedereen zich veilig voelt, en verbeter de gedragscode en de handhaving op basis
  die feedback.
</Paragraph>

## Community

Gebruik de Code of Conduct ook voor de open source community. Zorg voor respectvolle communicatie tussen een beperkt aantal "core contributors" met eigen prioriteiten, en een grote groep gebruikers met hoge verwachtingen en uiteenlopende belangen.

Vermijd druk vanuit de community om wijzigingen van onvoldoende kwaliteit door te voeren. Plaats het belang van een veilige werkomgeving en de belangen van de bredere community boven die van individuen.

## Zichtbaarheid van de Code of Conduct

Deel de Code of Conduct in elk geval in plekken waar wordt samengewerkt met de community, bijvoorbeeld in chatkanalen, in websites waar comments geplaatst kunnen worden en bij events die worden georganiseerd.

Bijvoorbeeld:

- Maak een `CODE_OF_CONDUCT.md` bestand in elke Git repository.
- Noem de Code of Conduct in de beschrijving van Slack-kanalen, en maak een bookmark naar de volledige tekst.

Gebruik de [`CODE_OF_CONDUCT.md` in de example repository](https://github.com/nl-design-system/example/blob/main/CODE_OF_CONDUCT.md) als voorbeeld.
