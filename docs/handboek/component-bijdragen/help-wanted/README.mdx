# Help Wanted

:::info In het kort

Dit component of deze variatie bestaat nog niet in code, maar we weten wel dat het nuttig zou zijn. De naam is bepaald, er is een beschrijving, en we hebben verzameld bij welke organisaties er behoefte aan is.

Help Wanted componenten zijn eigenlijk **een oproep**: wie wil dit component bouwen volgens de NL Design System aanpak?

:::
