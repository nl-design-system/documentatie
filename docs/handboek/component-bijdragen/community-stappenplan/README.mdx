# Stappenplan: Community

Voor de Community status zijn er twee stappenplannen:

1. [Het stappenplan voor organisaties](voor-organisaties) die een component willen bijdragen.
2. [Het stappenplan voor het kernteam](voor-kernteam) om een component de Candidate status te geven.

Er kunnen meerdere community implementaties zijn die volledig voldoen, deze staan allemaal op de [Community Componenten borden op GitHub](https://github.com/orgs/nl-design-system/projects?query=is%3Aopen+Community+Componenten).

## Heb je een vraag?

Heb je een vraag? Twijfel niet en stel je vraag via het [#nl-design-system Slack kanaal](https://praatmee.codefor.nl/) van CodeForNL of [neem contact op met het kernteam](/project/kernteam).
