# Nieuwe huisstijl met NL Design System

<Paragraph appearance="lead">
  Maak gebruik van onze tips om je dienstverlening te verbeteren, wanneer je een nieuwe huisstijl maakt. Door rekening
  te houden met gebruiksvriendelijkheid, toegankelijkheid en NL Design System.
</Paragraph>

## Alvast de checklist \{#checklist\}

- [ ] Gebruik moderne kleurcodes, bij voorkeur OKLCH
- [ ] Bied een volledig kleurenpalet aan, waarmee contrastproblemen worden voorkomen.
- [ ] Kies een lettertype die goed scoort op leesbaarheid.
- [ ] Deel stijlgids waarin problemen met contrast en leesbaarheid zijn opgelost.
- [ ] Maak een Design Tokens JSON bestand met al je ontwerpbeslissingen.
- [ ] Deel je ontwerp voor NL Design System componenten.
- [ ] Controleer je ontwerpbeslissingen door je Design Tokens te valideren.
- [ ] Wijs je leveranciers op bestaande NL Design System componenten.
- [ ] Wanneer jij of leveranciers Figma gebruiken: gebruik de NL Design System Figma met Design Tokens.
- [ ] Deel voorbeelden die voldoen aan NL Design System richtlijnen.
- [ ] Dark mode kleuren uitrekenen.
- [ ] Minimum en maximum tekstgrootte te kiezen, voor "fluid design".
- [ ] Tekstdikte kiezen.

Algemene checklist:

- Kleuren voor buttons, links en andere interactieve componenten kiezen
- Lettertype kiezen, eventueel extra lettertypes voor koppen en voor code.
- Signaalkleuren kiezen: rood, oranje, groen en blauw.
- Opties voor lettergrootte vastleggen.
