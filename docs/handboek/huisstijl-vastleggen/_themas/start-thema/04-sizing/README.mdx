# Sizing in het Start-thema

<Paragraph lead>
  Het Start-thema bevat een aantal tokens die gebruikt kunnen worden als elementen een bepaalde grootte moeten krijgen.
  Op deze pagina kun je deze tokens en hun waardes vinden.
</Paragraph>

Voor meer informatie over hoe deze tokens gebruikt worden kun je terecht in [het Start-thema handboek](/handboek/huisstijl/themas/start-thema).

## Algemene sizing-tokens

De algemene sizing-tokens kunnen worden gebruikt om alle soorten elementen een vaste grootte te geven en niet mee te laten groeien met de inhoud.

## Icon sizing-tokens

De icon-sizing tokens kunnen worden gebruikt om iconen een grootte te geven relatief tot de tekst eromheen.

## Pointer-target sizing-tokens
