# Form controls in het Start-thema

<Paragraph lead>
  Binnen de meeste thema's hebben formulierelementen zoals invoervelden, checkboxes en dropdowns veel overlap qua stijl.
  Ze gebruiken vaak dezelfde kleuren, borders en typografie voor dezelfde status (bv. focus, hover of invalid). Daarom
  defineert het Start-thema een aantal design tokens die gedeeld worden door de verschillende form-componenten. Op deze
  pagina staan al deze gedeelde tokens en waar ze voor gebruikt worden.
</Paragraph>

Voor meer informatie over hoe deze tokens gebruikt worden kun je terecht in [het Start-thema handboek](/handboek/huisstijl/themas/start-thema).
