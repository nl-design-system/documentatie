## Border-color tokens

Alle beschikbare kleuren zijn verwerkt in de [algemene kleurtokens](../kleuren) dus er zijn geen aparte tokens voor border-color. Als je een kleur wil kiezen voor een border bepaal je eerst in welke kleurcategorie (bv. `default`, `disabled`, `action-1`, etc) het element waar de border op komt het beste past. Als je een categorie gekozen hebt kun je daar een van de border tokens uit pakken (`border-subtle`, `border-default`, `border-hover`, `border-active`).
