# Start-thema

Vanuit NL Design System bieden we het 'Start-thema' aan. De stijl van dit Start-thema hebben we vastgelegd in een set aan ['basis-tokens'](/handboek/huisstijl/basis-tokens). Deze basis-tokens leven op het ['Common' niveau](/handboek/huisstijl/design-tokens#common-tokens).

Op deze pagina lees je wat het Start-thema is en welke stijlkeuzes er zijn gemaakt.

## Doel van het Start-thema

Met het Start-thema bieden we organisaties een stevig 'startpunt'. Het thema bevat toegankelijke en gebruiksvriendelijke standaarden, zodat je stap voor stap de huisstijl van jouw organisatie kunt doorvoeren. Zo kun je direct aan de slag met een consistente en inclusieve huisstijl, zonder alles zelf op te hoeven bouwen.

![Stijl van het Start-thema: een donkerblauw vlak met witte letters ‘Start-thema. Eenvoudig en neutraal.’ Rechts een illustratie van Nederland in dezelfde kleur.](https://raw.githubusercontent.com/nl-design-system/documentatie/assets/img_start-thema_stijl.png)

## Stijl van het Start-thema

De stijl is bewust eenvoudig en neutraal. Toch zijn er duidelijke keuzes gemaakt voor typografie, kleuren en iconen.

### Typografie

Als lettertype is gekozen voor IBM Plex Sans, een open source-lettertype dat voldoet aan onze [richtlijnen voor typografie](/richtlijnen/stijl/typografie/). Leuk detail: het is ontworpen door Mike Abbink van het Nederlandse bureau Bold Monday.

![De tekst IMB Plex Sans wordt afgebeeld met het lettertype IBM Plex Sans.](https://raw.githubusercontent.com/nl-design-system/documentatie/assets/img_start-thema_typografie.png)

### Kleur

De kleuren in het Start-thema bestaan uit blauw- en grijstinten, aangevuld met signaalkleuren. Alle combinaties voldoen aan de [richtlijnen voor kleur](/richtlijnen/stijl/kleuren/). Denk daarbij aan voldoende contrast voor [tekstkleur ten opzichte van de achtergrondkleur](/wcag/1.4.3). Maar ook voldoende contrast voor [niet-textuele content](/wcag/1.4.11).

Ook is er direct een dark-mode beschikbaar.

![Twee cirkels. De linker is donkerblauw de rechter is donkergrijs.](https://raw.githubusercontent.com/nl-design-system/documentatie/assets/img_start-thema_kleur.png)

### Iconen

#### Functionele iconen

![Tabler logo met op de achtergrond allerlei verschillende iconen uit deze set.](https://raw.githubusercontent.com/nl-design-system/documentatie/assets/meedoen_designers_voorbeeld-thema_functionele-iconen.png)

Voor functionele iconen gebruiken we een selectie uit de [Tabler iconenset](https://tabler-icons.io/). Deze open-source set bevat meer dan 3100 outline iconen en mag ook binnen jouw organisatie worden gebruikt.

Let op: deze set wordt niet door NL Design System beheerd. Je bent zelf verantwoordelijk voor het gebruik. Mis je een icoon? Dan kun je deze zelf toevoegen.

Aangezien de originele iconen uit lijnen van 2px bestaan die niet meeschalen met het formaat, hebben we ze omgezet naar vaste vormen (outline stroke). Dit sluit aan bij onze [richtlijnen rondom iconen](/richtlijnen/stijl/iconen).

#### Toptaak iconen

![Gemeente iconen logo met op de achtergrond allerlei verschillende iconen uit deze set.](https://raw.githubusercontent.com/nl-design-system/documentatie/assets/meedoen_designers_voorbeeld-thema_toptaak-iconen.png)

Voor toptaak iconen maken we gebruik van de outline-versie van de [OpenGemeenten iconenset](https://www.gemeenteniconen.nl/). Deze set met meer dan 200 iconen is open source en mag je vrij gebruiken.

Ook deze set valt buiten ons beheer. Mis je een icoon? Neem dan contact op met [OpenGemeenten](https://www.gemeenteniconen.nl/doe-mee).

### Voor designers

Wil je zien hoe het Start thema is toegepast op componenten? Neem dan gerust een kijkje in de [NL Design System Figma Bibliotheek met het Start thema](https://www.figma.com/design/FqAr99wvrlHxTJYAHkFRQN/NL-Design-System---Bibliotheek?node-id=197-664&t=t25LHoMjEkUZyEi1-1).

Je kunt deze bibliotheek gebruiken om zelf prototypes te maken of als startpunt om je eigen thema op te bouwen. Lees de [introductie voor designers](/handboek/designer/introductie) of volg het [Figma stappenplan](/handboek/designer/figma-stappenplan) om direct aan de slag te gaan.
