# Sizing in het Start-thema

Het Start-thema bevat een aantal tokens die gebruikt worden als elementen een minimale of vaste grootte moeten
krijgen. Op deze pagina kun je deze tokens en hun waardes vinden. Voor meer informatie over deze tokens kun je terecht in [het Start-thema handboek](/handboek/huisstijl/themas/start-thema).

## Algemene sizing-tokens

De algemene sizing-tokens worden gebruikt om elementen een vaste grootte te geven en niet mee te laten groeien met de inhoud.

## Icon sizing-tokens

De icon-sizing tokens worden gebruikt om iconen een grootte te geven relatief tot de tekst eromheen.

<CommonDesignTokensTable type="size" item="icon" />

## Pointer-target sizing-tokens

De pointer-target tokens worden gebruikt om te zorgen dat interactieve elementen zoals knoppen een minimale grootte krijgen.
