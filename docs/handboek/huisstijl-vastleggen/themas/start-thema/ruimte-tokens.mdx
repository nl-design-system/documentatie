# Basis-tokens voor ruimte

Deze pagina biedt een overzicht van de beschikbare basis-tokens voor ruimte ingevuld met waarden voor het Start-thema.

[Meer informatie over de basis-tokens en het Start-thema](/handboek/huisstijl/themas/start-thema).

## Tokens voor `block`-ruimte

Deze tokens worden toegepast voor ruimte haaks op de schrijf-richting.
Als de schrijf-richting horizontaal is (zoals bij Nederlands) worden deze tokens gebruikt voor verticale ruimte.
Als de schrijf-richting verticaal is (zoals bij Japans) worden ze gebruikt voor horizontale ruimte.
