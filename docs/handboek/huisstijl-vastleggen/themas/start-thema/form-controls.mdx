# Form controls in het Start-thema

Formulierelementen zoals invoervelden, checkboxes en dropdowns hebben vaak veel overlap qua stijl: ze gebruiken dezelfde kleuren, borders en typografie voor dezelfde status (bv. focus, hover of invalid).
Daarom defineert het Start-thema een aantal design tokens die gedeeld worden door de verschillende form-componenten.

Op deze pagina staan deze gedeelde tokens en waar ze voor gebruikt worden. Voor meer informatie over deze tokens kun je terecht in [het Start-thema handboek](/handboek/huisstijl/themas/start-thema).
