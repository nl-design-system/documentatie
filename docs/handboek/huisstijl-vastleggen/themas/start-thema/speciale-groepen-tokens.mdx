## Basis-tokens voor formulier-elementen

Formulier-elementen zoals invoervelden, checkboxes en dropdowns hebben vaak veel overlap qua stijl: ze gebruiken dezelfde kleuren, borders en typografie voor dezelfde status (bv. focus, hover of invalid).
Daarom defineert het Start-thema een aantal design tokens die gedeeld worden door de verschillende formulier-elementen.
