# Instructies voor documentatie schrijven voor basis tokens

## `_*-intro.md` bestanden

We hebben een verzameling teksten die uitleggen waar basis-tokens voor bedoeld zijn. De teksten worden gebruikt in de NL Design System website, maar ook in andere projecten waar uitleg over de tokens nodig is.

Schrijf een tekst die niet langer is dan 1 alinea. Gebruik geen links, omdat die niet gewenst zijn in andere applicaties. Schrijf voor een niet-technisch publiek. Gebruik niet de naam die de design tokens in code hebben, omdat de teksten ook gebruikt worden in applicaties die alleen een gebruiksvriendelijke user interface hebben zonder code.
