# Zelf een thema maken

De componenten van het NL Design System hebben van zichzelf geen specifieke huisstijl. Iedere organisatie kan zijn eigen huisstijl op de componenten toepassen. Om dat voor elkaar te krijgen maken we gebruik van ‘[design tokens](/handboek/huisstijl/design-tokens)’. We bieden we vanuit het NL Design System het '[Start-thema](/handboek/huisstijl/themas/start-thema/)' aan. De stijl van dit Start-thema hebben we vastgelegd in een set aan 'basis-tokens'. Deze basis-tokens leven op het ['Common' niveau](/handboek/huisstijl/design-tokens#common-tokens). Op basis van het [Start-thema](/handboek/huisstijl/themas/start-thema) hebben we ook een '[Voorbeeld-thema](/handboek/huisstijl/themas/voorbeeld-thema)' gemaakt. Op die manier kunnen we laten zien hoe je een eigen thema kunt doorvoeren. Het dient als 'voorbeeld'.

Wil je als designer een thema gaan maken voor jouw organisatie? Volg dan het [Figma Stappenplan](/handboek/designer/figma-stappenplan).
