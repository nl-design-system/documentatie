# Design Systems maken met NL Design System

Binnen het NL Design System ecosysteem kan elke organisatie makkelijk een eigen design system
starten. Organisaties gebruiken elkaars componenten, breiden deze uit en dragen nieuwe componenten
bij aan het ecosysteem.

Om snel te starten is er een GitHub template repository beschikbaar die aansluit bij de aanpak
van NL Design System. Design Tokens, Storybook en linting staan al klaar â€” zo kun je gelijk aan
de componenten beginnen.

Componenten bijdragen doe je via de stappenplannen onder de Definition of Done. Organisaties
kunnen daarnaast een eigen GitHub projectbord bijhouden om de voortgang van componenten in
ontwikkeling bij te houden.
