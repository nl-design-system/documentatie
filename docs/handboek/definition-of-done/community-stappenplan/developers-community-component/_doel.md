De component kan gedeeld worden voordat die helemaal klaar is, omdat het voor gebruikers makkelijk is om updates te installeren.

Developers hoeven de code van een component-implementatie niet te kopieeren om de component te gebruiken, omdat het makkelijker is de code via een package manager te installeren als dependency voor hun project.

Developers die npm kunnen gebruiken als package manager, kunnen de component installeren als dependency.
