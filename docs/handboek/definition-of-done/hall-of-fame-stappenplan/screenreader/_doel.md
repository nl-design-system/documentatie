Er is vastgelegd op welke manier de gebruikerservaring voor screenreader-gebruikers is geoptimaliseerd.

De kwaliteit van de screenreader-gebruikerservaring kan hiermee stabiel blijven, wanneer nieuwe versies van de component worden getest op de verwachte gebruikerservaring.

Er kan herkend worden dat de component aangepast moet worden, wanneer de ongewijzigde component minder goed werkt in nieuwe versies van browsers en hulpmiddelen, wanneer getest wordt met een nieuwe versie van de [NL Design System Baseline](./baseline).
