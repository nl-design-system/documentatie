De component voldoet in testscenario's aantoonbaar aan alle WCAG succescriteria die relevant zijn, voor zover de WCAG succescriteria onderdeel zijn van de [NL Design System Baseline](/baseline).

Het is duidelijk in welke testscenario's de WCAG succescriteria zijn getest.

Gebruikers kunnen vertrouwen dat alles is gedaan zodat de component toegankelijk ingezet kan worden in de situaties waarvoor de component bedoeld is.
