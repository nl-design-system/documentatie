De Candidate component kan makkelijk worden gebruikt in diverse projecten zodat er genoeg feedback wordt gegeven door developers om er uiteindelijk een Hall of Fame component van te maken.

Developers kunnen aan `@nl-design-system-candidate/` in de naam van de npm package herkennen dat de component nog niet helemaal stabiel is en dat deze is ontwikkeld door het NL Design System kernteam.
